## 📊 Возможности системы

### Клиент может:
- ✅ **Создавать заявки** на ремонт с выбором мастерской
- ✅ **Выбирать услуги** из списка (15 доступных услуг)
- ✅ **Просматривать свои заявки** со статусами
- ✅ **Отслеживать статус** заявки (new → in_progress → done)
- ✅ **Редактировать данные** в заявке (авто, описание, услуги)
- ✅ **Связаться с мастером** через контактные данные

### Мастер смены может:
- ✅ **Создавать заявки** от имени клиента в своей мастерской
- ✅ **Просматривать все заявки** своей мастерской
- ✅ **Менять статусы заявок** (new → in_progress → done)
- ✅ **Назначать техников** на заявки (только из своей мастерской)
- ✅ **Редактировать заявки** (авто, услуги, описание, данные клиента)
- ✅ **Удалять заявки** в статусах "Новая" или "В работе"
- ✅ **Просматривать техников** своей мастерской

### Администратор может:
- ✅ **Управлять мастерскими** (создание/редактирование/удаление)
- ✅ **Создавать пользователей** (мастеров и клиентов)
- ✅ **Привязывать пользователей к мастерским** (M2M связь)
- ✅ **Просматривать все заявки** со всех мастерских
- ✅ **Менять статусы любых заявок**
- ✅ **Назначать техников** из любой мастерской
- ✅ **Редактировать любые заявки** (включая данные клиента)
- ✅ **Удалять заявки в любом статусе** (включая "Готово")
- ✅ **Просматривать всех техников** и работников

## 📋 Статусы заявок

Система использует 3 статуса:

1. **new** (Новая) — заявка создана, ожидает обработки
2. **in_progress** (В работе) — заявка в работе, назначен техник
3. **done** (Готово) — заявка выполнена

**Правила смены статусов:**
- Админ может менять на любой статус
- При назначении мастера статус автоматически меняется с "new" на "in_progress"
- Удаление заявок: админ может удалять в любом статусе, мастер — только "new" и "in_progress"

---

## ⚠️ Важно!

- При первом запуске система **автоматически** создаст все начальные данные
- При повторном запуске данные **не будут созданы заново** (проверка по наличию пользователей)
- Для полного сброса используйте `docker-compose down -v`
- Техники привязаны к мастерским — нельзя назначить техника из другой мастерской
- При смене статуса показывается текущий техник (если назначен)

## 🎯 Как пользоваться?

1. **Открыть http://localhost:3000**
2. **Войти под любой учётной записью** (см. таблицу выше)
3. **Создать заявку:**
   - Кнопка "+ Создать заявку"
   - Выбрать мастерскую
   - Выбрать услуги
   - Описать проблему
4. **Отслеживать статус:**
   - new → in_progress → done
5. **Назначить техника:**
   - Открыть заявку
   - Нажать "Сменить статус"
   - Выбрать техника из списка (только из этой мастерской)
