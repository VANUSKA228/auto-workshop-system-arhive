# 🔧 Auto Workshop System

**Система управления автосервисом** — веб-приложение для автоматизации работы сети автомастерских.

---

## 📁 Версии проекта

| Версия | Описание | Статус |
|--------|----------|--------|
| [**v0-legacy**](v0-legacy/README.md) | Старая версия с полным функционалом | 📦 Legacy |
| [**v1**](v1/README.md) | Версия 1.0 — только авторизация и регистрация | ✅ Stable |
| [**v3-full**](v3-full/README.md) | Версия 3.0 — полная версия со всеми функциями | 🚀 Latest |

---

## 📋 Краткое описание версий

### v0-legacy
- Полный функционал: заявки, техники, мастера
- 7 мастерских в разных городах
- 3 роли: Клиент, Мастер смены, Администратор
- 15 услуг, 11 техников, 7 мастеров смены

### v1
- Только авторизация и регистрация
- JWT аутентификация
- Базовая структура проекта
- Идеально для начала разработки

### v3-full
- Полный функционал системы
- Создание и управление заявками
- Назначение техников
- Статусы заявок (new → in_progress → done)
- Все роли пользователей

## 🛠️ Технологический стек (общий для всех версий)

| Слой | Технология |
|------|------------|
| Backend | Python 3.11+, FastAPI, SQLAlchemy 2.0, Alembic |
| База данных | PostgreSQL 15 |
| Auth | JWT (python-jose, bcrypt) |
| Frontend | React 18, TypeScript, Vite, Tailwind CSS 3 |
| Deploy | Docker + Docker Compose |

## 🚀 Быстрый старт

Перейдите в папку нужной версии и выполните:

```bash
docker-compose up --build
```
